Frog Hop!
By: Ryan Funk

My game is a simple drag and hop platforming game where you try to get from one side of the river to the other.

The game features 3 Physics Based levels, and 3 Summary screens as well as an intro screen. 

The game tracks the players X and Y mouse coords when clicking on the frog an dragging back, inverts the coordinates to draw a line in the direction of travel, then moves the frog to the location based on the vector created.
